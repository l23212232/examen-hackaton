Examen U3: Página web para la administración de dispositivos hospitalarios.
Esta página web se realizo con la finalidad de la gestión y control de instrumentos hospitalarios, sirviendo como base de datos donde se registra e inicia sesión con roles asignados ( administrador, asistente, auditor) para realizar las funcionalidades básicas de una base de datos: modificar, buscar, eliminar y agregar.
